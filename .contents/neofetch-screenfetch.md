---

> neofetch telepítése debian rendszereken:

```
sudo apt install neofetch
```

> neofetch telepítése arch rendszereken:

```
sudo pacman -S neofetch
```

> automatikus indítás beállítása:

```
sudo su
```

```
echo neofetch >> /etc/bash.bashrc
```

> vagy (pl. manjaro-nál)

```
echo neofetch >> ~/.zshrc
```
